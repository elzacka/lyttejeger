# Personvernerklæring for Lyttejeger

_Sist oppdatert: 28. januar 2026_

## Kort oppsummert

Lyttejeger samler ikke inn personopplysninger. Appen kjører i nettleseren din og lagrer kun dine preferanser lokalt på din egen enhet.

- Ikke navn, e-post eller andre personopplysninger
- Bruker ikke informasjonskapsler (cookies) for sporing
- Ingen brukerkonto eller innlogging nødvendig

### Lokal lagring

Appen lagrer følgende lokalt i nettleseren din (IndexedDB):

- Avspillingsposisjon (hvor langt du har hørt i hver episode)
- Spillekø (episoder du har lagt i kø)
- Abonnementer (podcaster du følger)
- PWA-cache for offline-bruk

Denne informasjonen forlater aldri enheten din og kan slettes ved å tømme nettleserdata.

## Tredjepartstjenester

### Podcast Index API

Appen bruker [Podcast Index API](https://podcastindex.org) for å vise podcast-data. Når du søker etter podcaster:

- Søketeksten din sendes til Podcast Index sine servere
- IP-adressen din er synlig for deres servere (som med alle nettsider)
- Se [Podcast Index sin personvernerklæring](https://podcastindex.org/privacy) for detaljer

### GitHub Pages

Appen hostes på GitHub Pages. GitHub kan logge:

- IP-adresser
- Nettlesertype
- Tidspunkt for besøk

Se [GitHub sin personvernerklæring](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement) for detaljer.

## Rettighetene dine

Siden appen ikke samler inn personopplysninger, er det ingen data å:

- Be om innsyn i
- Be om sletting av
- Be om retting av

For lokalt lagrede data kan du:

- Slette alt via nettleserens innstillinger
- Bruke privat/inkognito-modus for å unngå lokal lagring
