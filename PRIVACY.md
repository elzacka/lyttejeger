# Personvernerklæring for Lyttejeger

*Sist oppdatert: Desember 2025*

## Kort oppsummert

Lyttejeger samler ikke inn personopplysninger. Appen kjører i nettleseren din og lagrer kun dine preferanser lokalt på din egen enhet.

## Hva vi samler inn

### Ingenting personlig
- Vi samler ikke inn navn, e-post eller annen personlig informasjon
- Vi bruker ikke informasjonskapsler (cookies) for sporing
- Vi har ingen brukerkontoer eller innlogging

### Lokal lagring
Appen kan lagre følgende lokalt i nettleseren din (localStorage):
- Dine filterinnstillinger
- Søkehistorikk (kun på din enhet)
- PWA-cache for offline-bruk

Denne informasjonen forlater aldri din enhet og kan slettes ved å tømme nettleserdata.

## Tredjepartstjenester

### Podcast Index API
For å vise podcast-data bruker vi [Podcast Index API](https://podcastindex.org). Når du søker etter podcaster:
- Din søketekst sendes til Podcast Index sine servere
- Din IP-adresse er synlig for deres servere (som med alle nettsider)
- Se [Podcast Index sin personvernerklæring](https://podcastindex.org/privacy) for detaljer

### GitHub Pages
Appen hostes på GitHub Pages. GitHub kan logge:
- IP-adresser
- Nettlesertype
- Tidspunkt for besøk

Se [GitHub sin personvernerklæring](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement) for detaljer.

## Dine rettigheter

Siden vi ikke samler inn personopplysninger, er det ingen data å:
- Be om innsyn i
- Be om sletting av
- Be om korrigering av

For lokalt lagrede data kan du:
- Slette alt via nettleserens innstillinger
- Bruke privat/inkognito-modus for å unngå lokal lagring
